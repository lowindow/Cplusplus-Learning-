# 一个空类占多少字节

```c++
class A {
    
};

int main(){
    A a;
    std::cout << sizeof(a) << std::endl;             # 1
    A a1;
    std::cout << &a << " " << &a1 << std::endl;      # 地址不一样
}
```

空类大小为1字节，占位符，不存储任何数据，用于占据内存空间，区分不同对象的内存地址，
如果是0的话，无法区分。

注意：

派生空类 编译器会做空基类优化 派生类大小仍为1字节 避免冗余

包含成员的类 1字节占为会被覆盖，具体遵循内存对齐规则，取决于具体成员大小。
